## 使用方法

### 1. 克隆项目
```bash
git clone https://github.com/seiuna/pukoudai-client.git
```

### 2. 安装依赖
```bash
npm install
```

### 3. 运行
```bash
npm run transfer # 将一个活动转给另一个人
npm run clicker # 当活动到达报名时间时自动报名
npm run monitor # 监控活动如果有人退出则自动加入 
```
